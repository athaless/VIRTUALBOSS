VirtualBOSS
Previsões para as próximas versões


Nosso objetivo aqui é descrever as funcionalidades nas quais estamos trabalhando para as próximas versões do VirtualBOSS, sistema que hoje abrange uma série de funcionalidades que o tornam extremamente flexível a vários tipos de aplicação e uso que vão desde operações ERP, CRM, Controle de Projetos até HelpDesk.

Não entraremos em detalhes para manter sigilo sobre os novos lançamentos.

Funcionalidades

  • Encapsulamento do núcleo e demais rotinas em classes
  • Implementação de NFe de serviço para Pref. SP / NFe de Produtos (uso em SP e SP)
  • Novo sistema de pedidos/produtos e serviços com implementação extra de Painel (como o de chamados) para solicitação/pedidos direto de clientes e visualização do seus pagamentos, reemissão de boletos, etc)
  • Complemento aos módulos de Biblioteca - controle de empréstimos e devoluções
  • Definição de protocolo para disponibilização de dados a sistemas de terceiros
  • Ajuste finalizando os módulos de gestão de Cargos/Cargos e Competêndias)

Otimizações

  • Conversão de rotinas de script para storedprocedures
  • Estudo de mudança de DB (MYSQL5 to PostgreSQL) / uso de replicação de dados
  • Integração com API Google e possibilidade sub-módulos ligados ao Facebook

Help

  • Manual geral e básico dos módulos do sistema
  • Manual de desenvolvimento, regras e frameworks para novos desenvolvedores
